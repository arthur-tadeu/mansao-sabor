# Mansão Sabor

Project developed as a full stack study practice.

The application was built to consolidate frontend and backend fundamentals, including:

* Interface structuring
* API integration
* Data manipulation
* Code organization

Main objective: to improve technical skills through the development of a functional system, simulating a real-world application scenario.

This project is not intended for commercial use and is focused исключительно on learning, experimentation, and continuous improvement.
